# Meme KOLs 激励平台

## 项目简介
这是一个为 Web3 KOL 量身定制的激励系统平台，支持通过专属链接追踪用户交易并分发奖励。

### 主要特性
- Web3 钱包登录（Solana）
- KOL 数据看板
- 邀请链接追踪
- 交易数据统计
- 奖励分发管理

### 技术栈
- Next.js 14
- Shadcn UI
- TailwindCSS
- Lucid Icons
- Solana Web3.js

## 开始使用

```bash
# 安装依赖
npm install

# 运行开发环境
npm run dev

# 构建生产环境
npm run build
```

## 项目结构
```
src/
  ├── app/              # Next.js 14 应用目录
  ├── components/       # React 组件
  ├── lib/             # 工具函数和配置
  ├── hooks/           # 自定义 Hooks
  └── types/           # TypeScript 类型定义
```
